# excelPainter

🎈软件用于在**excel**中绘制像素画，仅支持个人娱乐开发交流，禁止用于个人盈利。

🎨**1000*700**像素图片绘制大约需要`36s`

## 环境依赖

- python3
- PyQt5
- openpyxl
- pillow
